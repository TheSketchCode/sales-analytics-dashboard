# 📊 Sales Analytics Dashboard

## 📌 Project Overview
This project is an Excel-based Sales Dashboard created to analyze sales performance, profit trends, and customer insights.

## 📊 Features
- Year-wise Sales and Profit Analysis
- Top 5 Customers by Profit
- Category-wise Sales Distribution
- Interactive Dashboard using Pivot Tables and Charts

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Data Visualization
